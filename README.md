# Installing
To get the project running, follow these steps:
- Create a folder
-Enter/Navigate into that folder
```
cd <folder-name>
```
- Install all the project's dependencies:
```
yarn or npm install
```
- Once that's done, run the project:
```
npm start or yarn start
```
